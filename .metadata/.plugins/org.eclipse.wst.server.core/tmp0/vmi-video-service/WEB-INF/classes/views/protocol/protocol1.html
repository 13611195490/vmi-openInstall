4.2、特别说明

4.2.1、总额构成

总额是指用户实际支付的充值本金、通过平台赚取的订单佣金及蜜聊平台各种形式奖励现金，这些总和会构成用户的账户总额。

总额有效期为自进入蜜聊账户之日起直至消费完为止;可提现部分直至审批提现完为止。

4.2.3、总额使用规则

总额包括用户收入和用户充值两部分，平台一切奖励（除登录奖励的现金红包外）及充值均划归入“我的充值”账户，可用作支付包括但不限于诚意金、定金及全款，但不能转赠及提现。“我的收入”为用户订单赚取佣金，符合提现条件用户可申请提现。

用户使用期间会优先扣除蜜聊账户中“我的充值”。当用户未在平台上产生订单时，不可退回账户充值金额。若用户已发生平台订单，优先扣除用户充值金额，“我的充值”中所剩余额可做在平台其他消费使用。“我的收入”的金额可在用户状态正常情况下认证本人身份证件准确无误后进行申请提现，并同意扣除手续费的，申请提现后，经过平台核实无问题资金会在用户申请之日起一周内审批提现。
